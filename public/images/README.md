# Imagens dos Projetos

Adicione suas imagens nesta pasta com os seguintes nomes:

## Imagens Necessárias

- `zabbix-grafana.jpg` - Screenshot do dashboard Zabbix/Grafana
- `whatsapp-alerts.jpg` - Screenshot dos alertas via WhatsApp
- `incident-dashboard.jpg` - Screenshot do painel de incidentes

## Formato Recomendado

- Formato: JPG ou PNG
- Largura mínima: 1200px
- Proporção: 16:9 ou 4:3
- Tamanho: Menos de 2MB por imagem

## Como Adicionar

1. Copie suas imagens para esta pasta
2. Renomeie-as com os nomes acima
3. As imagens aparecerão automaticamente na página
